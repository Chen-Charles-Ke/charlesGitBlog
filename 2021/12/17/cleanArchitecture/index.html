Its a pretty easy requirement yeah ? i mean like sample bussiness, sample response data. So the qustion is how will you stcuture your folder with Clean Architecture?

Folder Overview

Data

  • Mappers —> data mapper
  • Repositories —> network call impl

Domain

  • Entities —> all view models
  • Repositories —> network call protocols
  • UseCases —> feature useCase

UI

  • View
  • ViewModel
  • ViewController (if needed, demo in this article based on pure MVVM so we actually will not have VC)
